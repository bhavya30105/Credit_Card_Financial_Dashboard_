# Credit_Card_Financial_Dashboard_
Credit Card Financial Dashboard Project
Project Overview
This project involves the development of a comprehensive Credit Card Weekly Dashboard that provides real-time insights into key performance metrics. The goal was to monitor transaction trends, customer demographics, and revenue growth across 53 weeks of data.

Tech Stack
Database: MySQL (Data Storage and Management)

Visualization: Power BI Desktop

ETL: Power Query and SQL

Analysis: DAX (Data Analysis Expressions)

Key Technical Milestones
Database Integration: Successfully imported and managed 10,000+ rows of transaction and customer data into a MySQL local server.

Advanced ETL & Data Cleaning: Resolved complex data issues, including BOM encoding characters in CSV files and dynamic date formatting using Locale-based transformations.

Data Modeling: Developed a robust One-to-Many relationship between Customer and Credit Card tables to enable cross-filtering and deep-dive analysis.

Interactive Dashboards: Designed a 2-page dashboard featuring Revenue by Age-Group, Education Type, Job, and an advanced QTR Revenue vs. Transaction Volume trend chart.

Business Metrics Calculated
Total Revenue: $57M

Total Interest: $7.98M

Transaction Count: 667K

Revenue Growth: Integrated Week-53 data to update annual performance metrics.
